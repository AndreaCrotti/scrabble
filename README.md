# scrabble

Various functions to generate list of valid words for scrabble/words with friends.

## Installation

Download from https://github.com/AndreaCrotti/scrabble

## Usage

    $ lein run -w <word>

## Examples

    $ lein run -w friend
    ({:value 11, :word finder} {:value 11, :word friend})
